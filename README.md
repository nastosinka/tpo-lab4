# Установка Jmeter на mac

1) Скачать binaries .tgz с официального сайта (https://jmeter.apache.org/download_jmeter.cgi). 
2) ```tar -xzf apache-jmeter-*version*.tgz``` - распакуйте и установите
3) ```sudo mv apache-jmeter-*vesrion* /usr/local/jmeter``` - переместите в /usr/local, чтобы можно было вдальнейшем использовать как команду пользователя
4) ```sudo ln -s /usr/local/jmeter /usr/local/bin/jmeter``` - сделайте symlink для вызова из любого места
5) ```jmeter``` - зпустите GUI
